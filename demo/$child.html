$child()
$child()方法根据DOM的id获得DOM节点的子节点。
$child()方法返回的结果为一个或多个DOM节点对象，如果指定的标签名称不存在任何节点，则返回null。
页面中有以下HTML 
	<div id="parent">
		<span id="child1">Demo Child HTML 1</span>
		<span id="child2">Demo Child HTML 2</span>
	</div>
以下代码可以获得id为parent的所有子节点 
	var child_obj = $child("parent");
演示代码： 
	<div id="parent">
		<span id="child1" style="background-color:#999;padding:3px;">Demo Child HTML 1</span>
		<span id="child2" style="background-color:#999;padding:3px;">Demo Child HTML 2</span>
	</div>
	<div id="output"></div>
	
	<script language="javascript" type="text/javascript">
	var child_obj = $child("parent");
	
	$("output").innerHTML += "Child DOM id is : ";
	for (var i=0; i<child_obj.length; i++) {
		$("output").innerHTML += child_obj[i].id + ", ";
	}
	</script>
