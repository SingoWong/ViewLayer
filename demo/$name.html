$name()
$name()方法等效于document.getElementsByName()，为了缩短开发中输入的代码量，ViewLayer提供了简洁的方式根据DOM的name获得DOM节点。
$name()方法返回的结果为一个或多个DOM节点对象，如果指定的name不存在任何节点，则返回null。
页面中有以下HTML 
	<div id="panel">
		<span><input type="checkbox" name="chkDemo" value="Demo1"/></span>
		<span><input type="checkbox" name="chkDemo" value="Demo2"/></span>
		<span><input type="checkbox" name="chkDemo" value="Demo2"/></span>
	</div>
以下代码可以获得所有name为chkDemo的节点 
	var obj = $name("chkDemo"); //得到的obj长度为3
该方法等效于 
	var obj = document.getElementsByName("chkDemo");
演示代码： 
	<span><input type="checkbox" name="chkDemo" value="Demo1"/></span>
	<span><input type="checkbox" name="chkDemo" value="Demo2"/></span>
	<span><input type="checkbox" name="chkDemo" value="Demo3"/></span>
	<div id="output"></div>
	
	<script language="javascript" type="text/javascript">
	var obj = $name("chkDemo");
	
	for (var i=0; i<obj.length; i++) {
		$("output").innerHTML += obj[i].value + "</br>";
	}
	</script>
