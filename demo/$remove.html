$remove()
$remove()方法根据DOM的id移除DOM节点。
页面中有以下HTML 
	<div id="parent">
		<span id="child1">Demo Child HTML 1</span>
		<span id="child2">Demo Child HTML 2</span>
	</div>
以下代码可以移除id为child2的节点 
	$remove("child2");
演示代码： 
	<div id="parent">
		<span id="child1" style="background-color:#999;padding:3px;">Demo Child HTML 1</span>
		<span id="child2" style="background-color:#999;padding:3px;">Demo Child HTML 2</span>
	</div>
	
	<script language="javascript" type="text/javascript">
	$remove("child2");
	</script>
