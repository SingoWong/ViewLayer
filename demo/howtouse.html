如何使用
下载ViewLayer文件压缩包后，解压放入目标站点中（放置位置可随意），在需要使用ViewLayer的页面中引用ViewLayer目录中的core.js文件。 
	<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
    <html xmlns="http://www.w3.org/1999/xhtml">
	<head>
	<title>Hello ViewLayer</title>
	<script language="javascript" type="text/javascript" src="viewlayer/core.js"></script>
    </head>
	...
添加应用后即可在该页面中使用ViewLayer的方法。 
	<script language="javascript" type="text/javascript">
    ViewLayer.onReadu(function(){
		alert("Hello ViewLayer");
	});
	</script>
