如何使用
ViewLayer准备了一些常用的方法

ViewLayer.onReady()
功能：页面所有DOM元素都加载完成后执行脚本。
当您有一些Javascript脚本需要在页面加载完成时执行，但又担心脚本放在<head></head>中DOM还没完全加载完成时被执行，则可以使用ViewLayer.onReady方法，确保页面中所有DOM元素都加载完成后执行。 

	ViewLayer.onReady(funtion(){
		//这里放入你要执行的代码
		document.write("Hey，所有的DOM都加载完成了哦。");
	})

ViewLayer.onAllReady()
功能：页面所有元素（包括DOM/图片等）都加载完成后执行脚本。
该方法和ViewLayer.onReady相似，但该方法则是待页面中所有包括图片，多媒体元素都加载完成后才执行。 

	ViewLayer.onAllReady(funtion(){
		//这里放入你要执行的代码
		document.write("Hey，所有的元素都加载完成了哦。");
	})

ViewLayer.Include(path, file)
功能：加载外部脚本。
参数：path为要加载的脚本文件的目录，file为要加载的脚本文件文件名
使用ViewLayer加载一个js脚本文件。 

	ViewLayer.Include("script", "hello.js"); //这里加载了一个script目录下的hello.js脚本文件

ViewLayer.getBrowser()
功能：获取当前用户所使用的浏览器。
获得当前用户所使用的浏览器类型，可能出现的结果为msie, firefox, opera, chrome 

	if (ViewLayer.getBrowser() == "msie") {
    	
    	alert("现在用的是Microsoft Internet Explorer");
        
    } else if (ViewLayer.getBrowser() == "firefox") {
    	
    	alert("现在用的是Mozilla Firefox");
        
    } else if (ViewLayer.getBrowser() == "opera") {
    	
    	alert("现在用的是Opera");
        
    } else if (ViewLayer.getBrowser() == "chrome") {
    	
    	alert("现在用的是Google Chrome");
        
    } else{
    	
    	alert("现在用的是未知浏览器");
        
    }

ViewLayer.getPosTop(), ViewLayer.getPosLeft()
功能：获取指定元素在页面的横坐标和纵坐标

Tips:以下例子中使用了ViewLayer的DOM方法$()，详细说明请参阅DOM $()

    var obj = $("panel"); //获取id为panel的元素
    var x = ViewLayer.getPosLeft(obj);  //获得该元素在页面中的横坐标
	var y = ViewLayer.getPosTop(obj);   //获得该元素在页面中的纵坐标
