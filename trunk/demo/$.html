$()
$()方法等效于document.getElementById()，为了缩短开发中输入的代码量，ViewLayer提供了简洁的方式根据DOM的id获得DOM节点。
$()方法返回的结果为一个DOM节点对象，如果指定的id不存在，则返回null。
页面中有以下HTML 
	<div id="panel">
		<span>Demo HTML</span>
	</div>
以下代码可以获得id为panel的节点 
	var obj = $("panel");
该方法等效于 
	var obj = document.getElementById("panel");
演示代码： 
	<div id="panel" style="font-weight:bold;border:1px solid #CCC;width:100px;padding:5px;">
    	<span>Demo Panel</span>
    </div>
    <div id="output"></div>
                                    
    <script language="javascript" type="text/javascript">
    var obj = $("panel");
                                            
    for (e in obj) {
        $("output").innerHTML += e + " = " + obj[e] + "<br/>";
    }
	</script>
