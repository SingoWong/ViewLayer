$parent()
$parent()方法根据DOM的id获得DOM节点的父节点。
$parent()方法返回的结果为一个DOM节点对象，如果指定的标签名称不存在任何节点，则返回null。
页面中有以下HTML 
	<div id="parent">
		<span id="child">Demo Child HTML</span>
	</div>
以下代码可以获得id为child的父节点 
	var parent_obj = $parent("child");
演示代码： 
	<div id="parent">
		<span id="child">Demo Child HTML</span>
	</div>
	<div id="output"></div>
	
	<script language="javascript" type="text/javascript">
	var parent_obj = $parent("child");
	$("output").innerHTML += "Parent DOM id is : " + parent_obj.id;
	</script>
